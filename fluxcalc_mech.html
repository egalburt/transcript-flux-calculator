TRANSCRIPT FLUX CALCULATOR 3: Regulation Across Promoter Space

This calculator explores the phase space of gene regulation as a function of basal (unregulated) rates of transcription initiation. Unregulated rate constants (1/s) and polymerase concentration ([R], nM) are enetered below in the left column. The fold-change of each parameter generated upon regulation is in the second column and defines a fixed regulatory mechanism. Two basal parameters are selected using the drop-down menus. The X- and Y- parameters will be titrated geometrically from their input value to their input value times the X- and Y-multiplier respectively using the number of points. When the "GO" button is clicked, the mRNA flux fold-change for the defined regulatory mechanism applied to the defined set of basal parameters is calculated. The resulting plot is color coded to indicate regions of promoter space that will be activated (flux ratio > 1, green) or repressed (flux ratio < 1, red). Data can be downloaded by clicking on the image of the disk over the plot and an image may be downloaded by clicking on the camera or taking a screen shot.
